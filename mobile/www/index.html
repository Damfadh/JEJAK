JEJAK Scanner (PWA)

Scan QR halftone dari dokumen dan verifikasi ke backend.

Persetujuan penggunaan aplikasi

Sebelum memakai scanner, Anda harus menyetujui:

  • Akses kamera untuk memindai QR halftone pada dokumen.
  • Pengiriman data hasil scan ke backend JEJAK untuk verifikasi.
  • Penyimpanan konfigurasi API Base URL dan API Key secara lokal di perangkat ini.
  • Penggunaan hanya untuk dokumen yang berwenang/diizinkan.

Jika Anda tidak setuju, aplikasi tidak dapat digunakan.

Scan a QR code on the document.
Tip: gunakan cahaya cukup dan posisikan QR dekat kamera saat mode halftone sangat samar.
