# Template Entry — PROGRESS

Gunakan template ini untuk semua entri baru (progress / issues / notes).

## Progress entry
- Tanggal: YYYY-MM-DD
- Author: @nama
- Judul singkat: 
- Status: open | in-progress | done
- Deskripsi: Jelaskan apa yang dilakukan (singkat, 2-4 baris)
- Files changed: list file path (jika ada)
- Next steps: langkah selanjutnya
- Notes: catatan tambahan

## Issue entry (singkat)
- ID: ISSUE-XXX
- Tanggal:
- Pelapor:
- Judul:
- Status:
- Prioritas:
- Deskripsi:
- Langkah reproduksi:
- Assigned:
- Solusi:
- Komentar:

Pastikan setiap entry diberi tanggal dan author.
